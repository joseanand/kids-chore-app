# Kids Chore App

Simple Spring Boot application for:
- Kids login
- Chore tracking
- Rewards redemption
- Parent admin management
- H2 in-memory database
- Hourly SQL backups

## Default Users

Parent:
- username: parent
- password: password

Kid:
- username: kid1
- password: password

## Run

```bash
mvn spring-boot:run
```

## H2 Console

http://localhost:8080/h2-console
